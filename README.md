# Tugas-citra-digital-menggunakan-metode-color_enhancement.py
Proyek ini merupakan tugas manipulasi citra digital dengan metode Color Enhancement, yaitu teknik untuk membuat foto klasik menjadi lebih berwarna, lebih cerah, dan terlihat seperti foto modern.
Metode yang digunakan adalah:
* HSV Color Boosting (Saturation Enhancement)
* Brightness Adjustment
* Sharpening & Light Smoothing
Hasil akhir berupa gambar yang lebih hidup dan berwarna natural.
